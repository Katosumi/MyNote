# MyNote

Personal Knowledge Base managed with [Obsidian](https://obsidian.md/).

## Overview
This repository contains my personal notes and knowledge base.
It is automatically synced using the [Obsidian Git](https://github.com/denolehov/obsidian-git) plugin.

## Structure
- **Notes**: Markdown files containing my notes.
- **Attachments**: Images and other media files.
- **.obsidian**: Configuration files for Obsidian (plugins, themes, etc.).

## Synchronization
- **Commit Interval**: 10 minutes (Local history)
- **Push Interval**: 30 minutes (Cloud backup)
- **Pull Interval**: 1 minute (Sync on startup/interval)

## Usage
1. Clone this repository.
2. Open the folder as a Vault in Obsidian.
3. Install the "Obsidian Git" plugin to enable sync.
